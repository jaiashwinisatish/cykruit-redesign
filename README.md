# 🚀 Cykruit Redesign 
A modern, high-performance redesign of the Cykruit website built using **React + Vite + TypeScript + Tailwind CSS**.
This project focuses on **clean architecture, premium UI/UX, and scalable frontend practices**.

Build a production-ready frontend web application by redesigning and rebuilding the website Cykruit.com from scratch using Next.js (App Router) and Tailwind CSS.

---

## ✨ Overview

This project reimagines the original Cykruit platform with:

* A **modern dark-first cyber aesthetic**
* Improved **user experience & accessibility**
* Scalable and maintainable **frontend architecture**

The goal is not just replication, but delivering a **next-generation product experience**.

---

## 🛠 Tech Stack

* **React** – UI Library
* **Vite** – Fast build tool
* **TypeScript** – Type safety
* **Tailwind CSS** – Utility-first styling
* **Framer Motion** – Animations
* **React Router DOM** – Routing
* **Zustand / Context API** – State management
* **React Hook Form + Zod** – Form validation

---

## 📂 Project Structure

```
src/

├── hooks/              # Custom hooks
├── lib/                # Utilities & helpers
├── store/              # Global state
├── types/              # Type definitions
└── assets/             # Images, icons, etc.
```

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/cykruit-redesign.git
cd cykruit-redesign
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```
